---

## 分类说明

### 业务域（一级分类）
- AI编程能力
- 组件库
- 脚手架
- 网站 / 官网
- 其他业务域...

### 模块（二级分类，可选）
用于复杂业务域下的细分：
- 表格组件
- 新组件开发
- 视觉与主题
- 工程化与发包

### 重点任务
- 具体的功能点或任务名称

### 具体内容
- 任务的详细描述

---

## 状态标记

| 状态 | 说明 |
|------|------|
| 计划中 / 待开始 | 任务尚未开始 |
| 进行中 / 开发中 | 任务正在进行 |
| 已完成 / 已验收 | 任务已完成 |
| xx% | 具体完成百分比 |

---

## 关注项与依赖

用于记录：
- 需要重点关注的问题
- 依赖其他团队/人员的事项
- 风险预警
- 延期事项说明
